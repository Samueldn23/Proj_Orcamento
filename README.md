# Aplicativo de Orçamento

## Descrição

Este aplicativo permite que os usuários gerenciem orçamentos.

## Instalação

1. Clone o repositório.
2. Instale as dependências:

   pip install flet

## Uso

Execute o aplicativo:
flet run app.py
